🌦️ Weather-Dashboard

A responsive and user-friendly **Weather Application** built using **HTML, CSS, and JavaScript** that provides real-time weather updates using API integration.

---

## 📌 Features

* 🌍 Search weather by city name
* 🌡️ Displays real-time temperature
* ☁️ Shows weather conditions (Cloudy, Sunny, Rainy, etc.)
* 📍 Displays location details
* 🎨 Clean and responsive UI
* ⚡ Fast API data fetching using JavaScript


---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** OpenWeather API
* **Tools:** VS Code, Git, GitHub
